### Pendahuluan

Hallo semuanya, pada kesempatan kali ini saya akan membagikan pengalaman mengenai saya pindah dari os windows ke linux dengan menggunakan distro Kubuntu.

### Alasan

Alasan saya pindah dari os windows ke linux menggunakan distro Kubuntu karena saya ingin mencoba suasana baru, dan juga karena dukungan untuk update Windows 10 sudah tidak lagi berjalan. Oleh karena itu pada akhirnya saya memilih jalan untuk pindah ke os linux menggunakan distro kubuntu.

### Kenapa memilih distro turunan ubuntu dibandingkan arch?

Alasan saya cukup sederhana, karena ubuntu cukup familiar untuk saya pada saat ini, kenapa karena dulu saya suka oprek hp menggunakan termux dan command nya sama, itulah alasan saya akhirnya memilih distro turunan ubuntu dibandingkan dengan distro turunan arch.
