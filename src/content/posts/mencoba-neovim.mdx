### Penduluan

Pada kesempatan kali ini saya punya akan membagikan pengalaman menarik menggunakan text editor Neovim

### Alasan saya mencoba Neovim

Alasan saya mencoba neovim cukup sederhana, karena menurut saya ini salah satu text editor di linux yang musti saya coba, dan alhasil saya mencoba text editor ini, namun karena saya masih pemula dalam hal ini, saya coba belajar dari internet bagaimana command untuk keluar, mengedit, dan block line di vim itu sendiri.
