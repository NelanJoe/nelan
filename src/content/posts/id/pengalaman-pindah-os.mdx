### Pendahuluan

Halo semuanya! Pada kesempatan kali ini, saya ingin membagikan pengalaman saya saat memutuskan untuk bermigrasi dari OS Windows ke Linux, khususnya menggunakan distro Kubuntu.

### Alasan Berpindah

Ada dua alasan utama mengapa saya akhirnya meninggalkan Windows dan beralih ke Kubuntu. Pertama, saya ingin mencoba suasana dan tantangan baru dalam menggunakan komputer. Kedua, mengingat dukungan pembaruan (_update_) untuk Windows 10 yang akan segera berakhir, saya merasa ini adalah waktu yang tepat untuk mencari alternatif sistem operasi yang lebih "long-term".

### Mengapa Memilih Turunan Ubuntu (Kubuntu), Bukan Arch?

Alasannya cukup sederhana: familiaritas.

Bagi saya, ekosistem Ubuntu terasa lebih akrab karena saya memiliki pengalaman mengutak-atik (_oprek_) Android menggunakan Termux. Perintah dasar (_command line_) di Termux sangat mirip dengan keluarga Debian/Ubuntu. Karena sudah terbiasa dengan perintah-perintah tersebut, saya merasa distro turunan Ubuntu adalah pilihan yang jauh lebih masuk akal dan ramah untuk memulai dibandingkan langsung terjun ke distro berbasis Arch.
